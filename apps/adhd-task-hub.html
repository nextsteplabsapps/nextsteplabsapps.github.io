Core promise

Less task-manager maintenance. More usable next actions.

The app is not a medical treatment or emergency tool. It is a personal planning aid designed to reduce friction around ordinary tasks, routines, reminders, and life admin.

Closed-test improvements

What changed after tester feedback.

01

First-interaction task creation hardened

The Quick Add checklist path now has stronger database readiness and list-existence protection so first-use task creation is safer.

02

Android keyboard behavior improved

Custom bottom chrome, action docks, and sticky ad areas hide while the keyboard is open, with resize-aware fallback handling for older Android devices.

03

Bottom sheets respect navigation bars

Icon pickers and add-item sheets now include safe-area padding so controls are not clipped by gesture or 3-button navigation.

04

Cleaner navigation and actions

Stack screens avoid duplicate header spacing, and list item actions now use a focused action surface instead of expanded inline clutter.

05

Settings has public-facing links

Privacy Policy, Terms of Use, Rate App, Share App, and App Version are visible from Settings for tester confidence.

Plus

Optional support, not a requirement for the basics.

Included core use

Tasks, lists, Today, Hotlist, starter templates, task details, reminders, local backups, and core organization are built for local use without requiring an account.

Plus features

Plus is designed for people who want more help: optional AI Plan Assist, premium themes, recurring tasks, and an ad-free app experience.

Privacy shape

Local-first where it matters most.

On-device by default

Tasks, lists, notes, subtasks, templates, reminders, settings, local backups, and optional task photos are stored on the device for core use.

Optional network services

Purchases, ads for free users, diagnostics, analytics, app sharing/rating links, and optional AI planning may use external services as described in the Privacy Policy.

AI Plan Assist

If a user chooses AI help, selected task or planning context may be sent to the Plan Assist backend and AI provider to generate suggestions. Users should review suggestions before acting.

For testers

Trying the closed-test build?

Please pay special attention to cold launch, Quick Add checklist creation, typing on input-heavy screens, bottom sheets with Android navigation bars, Settings links, rating, sharing, and displayed version information.
