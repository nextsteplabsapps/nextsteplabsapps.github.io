Privacy

Clear privacy, built around local storage.

ADHD Task Hub is designed so the core experience works on-device. This page explains the current privacy approach in plain language.

1. Core app data stays on your device

Tasks, lists, notes, subtasks, focus sessions, reminder settings, and templates are stored locally on your device. The app does not require an account to work.

2. Local reminders

If you enable notifications, the app can schedule local reminders on your device. These reminders are used to help you follow up on tasks you choose to track.

3. Backups are user-initiated

The app can export a local JSON backup file when you choose to do so. Importing a backup is a manual action and replaces the current local app data on the device.

4. Diagnostics in some builds

Some builds may include limited analytics or crash diagnostics to help improve reliability. These are configured to avoid task titles, task notes, and other user-written content wherever possible.

5. No cloud sync at this stage

ADHD Task Hub does not currently include account login, team collaboration, or automatic cloud sync for core app data.

6. Changes over time

If the app adds new services or data flows later, this privacy page will be updated before or alongside those changes.
