# Next Step Labs site

This folder is a static website prepared for GitHub Pages.

## Local preview

### Fastest option

Open [index.html](C:/adhd-task-app/site/index.html) directly in your browser.

### Better option with a local server

From `C:\adhd-task-app\site`, run one of these:

```powershell
py -m http.server 4173
```

or

```powershell
python -m http.server 4173
```

Then open:

```text
http://localhost:4173
```
